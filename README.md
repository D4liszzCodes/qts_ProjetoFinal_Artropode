### Projeto CRUD Testado de Ponta a Ponta com o tema "artropodes"

Esse projeto visa implementar um sistema de leitura, exclusão, criação e atualização de dados de artropodes, contendo nome, especie, habitat e tempo de vida como atributos
Além disso, o trabalho tem como objetivo colocar em prática os conteúdos visitados durante o ano inteiro na matéria de QTS (Qualidade e Teste de Software), por essa razão esse sistema possui integração as
ferramentas de teste, jest e sellenium, este utilizando do recurso de POM, juntamente a isso, é utilizado o modelo de TDD (Test-Driven Develepmoment) para garantir maior controle na construção da aplicação, que por
fim é integrada a uma API hospedada no (software a escolher)
