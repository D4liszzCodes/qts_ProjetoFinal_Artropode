### Projeto CRUD Testado de Ponta a Ponta com o tema "artropodes"

Esse projeto visa implementar um sistema de leitura, exclusão, criação e atualização de dados de artropodes, contendo nome, especie, habitat e tempo de vida como atributos
Além disso, o trabalho tem como objetivo colocar em prática os conteúdos visitados durante o ano inteiro na matéria de QTS (Qualidade e Teste de Software), por essa razão esse sistema possui integração as
ferramentas de teste, jest e sellenium para a realização de etsets unitários, este utilizando do recurso de POM, juntamente a isso, é utilizado o modelo de TDD (Test-Driven Develepmoment) para garantir maior controle acerca dos possiveis erros no desenvolvimento da aplicação,que por fim é integrada a uma API hospedada no (software a escolher)

## Guia de Locação e Instalação - utilize qualquer prompt de comando (cmd, git bashe, powersheel etc)

### após acessar a pasta do projeto inicialize o node com o comando *npm init*
### em seguida instale o express com o comando *npm install express --save*
### instale também o body-parser com o comando *npm install body-parser --save*
### para se comunicar com o firebase e o firestore rode o comando *npm install firebase-admin --save*
### instale o ejs para renderizar a página de listagem utilizando o comando *npm install ejs --save*
### para migrar o firebase para o banco de dados mocado rode o comando *npm i sqlite sqlite3*
### por fim para conduzir os testes unitários rode o comando *npx jest* , esse comando além de rodasr os teste também instala o jest automaticamente
